## GitHub词汇表



| 名称   | 解释                                                                                                                                         | 说明 |
| ------ | -------------------------------------------------------------------------------------------------------------------------------------------- | ---- |
| `etch` | 使用 git fetch 时，是将更改从远程存储库添加到本地工作分支，而无需提交它们。 与 git pull 不同，提取允许在将更改提交到本地分支之前查看该更改。 |      |
